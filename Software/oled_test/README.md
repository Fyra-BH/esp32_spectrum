## SPI master half OLED12864 example

 This code demonstrates how to use the SPI master half duplex mode to read/write a SD1306 OLED DRIVER
 
### Connections

GPIO_SCL_PIN    GPIO12
GPIO_SDI_PIN    GPIO14
GPIO_RST_PIN    GPIO27
GPIO_DC_PIN     GPIO26
GPIO_CS_PIN     GPIO25


### Notes

If you meet timeout issues, please check your connections.
